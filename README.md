# Loay Magdy Ibrahim Portfolio - Final Version

Ready-to-run React/Vite portfolio with:
- Profile images
- Project screenshots and galleries
- Custom SVG icons
- Built-in CV download link
- Pure CSS styling
- No Tailwind and no lucide-react dependency

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repo on Vercel.
3. Deploy with default Vite settings.
